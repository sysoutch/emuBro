# Theme Manager Reference

## Core Theme Controls

- Create, save, update, and delete custom themes
- Switch between built-in and custom themes
- Basic and extended editing modes

## Color and Typography

- Main surface colors, accent colors, borders, and status colors
- Font choices for body and heading presentation
- Optional high-contrast and inverted theme variants

## Effects and Background

- Logo/text effect controls (mode, speed, intensity, colors)
- Background image controls and blend/opacity options
- Header and panel styling that follows theme variables

## AI Integrations

- AI theme generation from prompt/mood/style
- Locale translation tooling with optional translation style hint

## Related UX Areas

- Theme and language controls in responsive header wrappers
- Theme toggles for invert colors and invert theme behavior
