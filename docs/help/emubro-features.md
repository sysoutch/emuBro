# emuBro Features Reference

This reference is used by both:
- the in-app Help Docs browser
- the Support LLM grounding pipeline

## Library

- Views: Cover, List, Table, Slideshow, Random, Focus
- Filters: platform, region/language, group/sort, search
- Grouping: optional "group same names"

## Tools

- Cover Downloader: PS1/PS2 serial-based download
- Memory Card Editor: inspect, copy, import/export, create empty cards
- BIOS Manager: list/add/open BIOS folders
- Monitor tools and custom external tool shortcuts

## Launcher Integrations

- Steam, Epic, and GOG discovery/import flows
- Launch policy supports launcher URI and configured executable paths

## Support Assistant

- Modes: Troubleshoot, General Chat, Help Docs
- Grounds answers with local platform configs, DB game/emulator state, and help docs
- Debug mode can show retrieval plan and grounding context preview

## Updates

- App update checks and installs
- Resources update checks for emubro-resources data
